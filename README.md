# The First One Hundred

The Capsules Founding Creators campaign deck. Hosted via GitHub Pages at [f100.capsules.ink](https://f100.capsules.ink).

Built on the same visual system as the [main Capsules deck](https://github.com/TheThirdRoom/deck) — Literata + DM Sans typography, the capsule-pill hero, dark/light alternating slides, scroll-snap navigation, and the terracotta accent system.

## Campaign

100 storytellers. A living gallery. A campaign that builds itself — in public, milestone by milestone, until the gallery fills and the founding cohort closes forever.

## Structure

- **20 slides** — scroll-snap, keyboard nav (arrows / space)
- Title capsule hero with video → Brief → Brand filter → Audience → Insight → Curation → The First 100 → Gallery → Vehicle → Channels → Flywheel → Codes → Perks → Milestones → Invitation → Where we begin → Plan on a page → 100 → CTA
- Same CSS design system as the parent deck
- Gallery grid renders 100 capsule slots (17 filled as "current state")
